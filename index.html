Справочник тегов

Файлы с расширением .html открываются в браузере

  • Назначение текста
  • Выделение текста
  • Графические элементы
  • Структура страницы

1. Теги для текста

  • p - абзац от англ. paragraph

  • h1..h6 - заголовки от англ. heading

  • ul/ol - нумерованный и маркированный списки от англ. unordered/ordered list
    li - пункты списков от англ. list item

  • br - тег переноса строки
  • a - ссылка от англ. anchor (якорь)
    • Подробнее

2. Теги для выделения текста

  • b - жирный

  • i - курсивный

  • del - зачёркнутый

  • u - подчёркнутый

  • mark - выделенный

3. Теги для графических элементов

  • hr - линия от англ. horizontal rule

  • img - картинка от англ. image

4. Теги, задающие структуру страницы

  • !DOCTYPE html - не тег, а инструкция для сайта

  • html - заключает в себя всё содержимое веб-страницы

  • head - для хранения служебной информации о странице

  • body - содержимое документа HTML

  • div - универсальный контейнер

  • section - задаёт раздел документа
